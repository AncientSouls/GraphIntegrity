AncientGraphSpreading

Automatic maintenance spreading of graph according other graphs.

Install

npm install --save ancient-graph-spreading

Tests

Tests can be started with comand npm install ancient-graph-spreading && cd ./node_modules/ancient-graph-spreading && npm run compile && npm test. For more information lern src/tests/index.js.
